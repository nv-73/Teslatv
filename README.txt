INSTALLATION

1. Installer Node.js

2. Ouvrir le terminal dans le dossier

3. Installer :
npm install

4. Lancer :
npm run dev

5. Déployer sur GitHub Pages :
npm run deploy

IMPORTANT :
Le repo GitHub doit s'appeler :
Teslatv

GitHub -> Settings -> Pages

Source :
Deploy from a branch

Branch :
gh-pages

Folder :
/ (root)
